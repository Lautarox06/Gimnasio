------ Descripción ------

Este proyecto fue hecho como trabajo final de la materia Programación II.
El objetivo fue aplicar los temas vistos durante el año usando Java, trabajando con clases, objetos, colecciones, interfaces, excepciones y estructuras básicas.

El sistema permite administrar los clientes de un gimnasio, pudiendo dar de alta, modificar o eliminar socios, cambiarles el plan y calcular sus cuotas. Todo se maneja desde un menú por consola.

------ Funcionalidades ------

Alta, baja, búsqueda y modificación de clientes

Cambio de plan (Básico, Full o Premium)

Cálculo de cuota 

Lista de clientes ordenada

Estadística de altas por mes
